详细设计方案 — 模块与界面

方案 B: MediaPipe + HuggingFace 预训练模型 + Streamlit 仪表盘

📁 项目结构

文件 职责 对外接口
face_detector.py人脸检测 — MediaPipe Face Detectiondetect_faces(image) → list[Face]
expression_recognizer.py表情识别 — HuggingFace 预训练模型recognize(face_img) → Expression[]
analyzer.py统计分析 — 计数/比例/主导表情/时间序列analyze(results) → dict
utils.py辅助函数 — 画框、CSV读写、图像预处理draw_boxes(), write_csv(), etc.
app.pyStreamlit 主界面 — 整合所有模块streamlit run app.py
